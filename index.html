How to Apply CSS to HTML

1. Create a CSS File in the same folder as HTML named "mystyle.css"

Here's an example of what could look like in external CSS stylesheet: 

              
                  h3 {
                  color: orange;
                  background-color: grey;
                  font-family: "cursive";
                  }

Now inside our HTML document we use the href attribute along with link
to reference the file in your file system:

              
                    <link rel="mystylesheet" href="mystyle.css">

rel defines the relationship between the<link> element
and the current document it referencing such as "mystyle.css"
We can also use adjusted paths to reference the CSS file: 

              
                    <link rel="mystylesheet" href="resources/mstyle.css">
                    <link rel="mystylesheet" href="other/resources/mystyle.css">

Referencing an external CSS Stylesheet in HTML would then look like this with the <link> element
located in the <head> section our our HTML document: 

              
                        <html>
                           <link rel="mystylesheet" href="mystyle.css">

                          <body>
                              <h1>Applying CSS Externally in HTML </h1>
                          </body;>
                        < /html>

How to Use JavaScript in a Webpage

1. Create a Java File in the same folder as HTML named "myjscript.css"

Using JavaScript doesn't appear in a web page as a result you need the <script> element
which has a src attribute 

        
              <script src="other/resources/myjscript.js"></script>

The src element like href
refers to the script file in your file system
either in same folder or using a relative path 

        
            <script>
              window.addEventListener("load"), () =>{
                console.log("This function is executed once the page is loaded");
              };
            </script>

In addition you can also use Javascript code between the <script> tag.
Linking external javascript allows you to be focused on your work,
write self-sufficient HTML, and structured JavaScript applications
